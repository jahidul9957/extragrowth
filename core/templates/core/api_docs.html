API Overview

Our API allows you to programmatically place orders, check statuses, and manage your balance. It is fully compliant with the Standard SMM API v2 structure, meaning you can easily connect it to PerfectPanel, SmartPanel, or your custom scripts.

RESTful Architecture

Supports both GET and POST requests.

JSON Responses

All responses are in clean JSON format.

Multi-Platform

Automate YouTube, Instagram & more.

POST

Add Order

Place a new order automatically. Required balance will be deducted immediately.

Parameter Type Req. Description
keystringYour API key
actionstringMust be add
serviceintService ID
linkstringTarget URL
quantityintAmount needed
 
curl -X POST https://{{ request.get_host }}/api/v2/ \
  -d "key=YOUR_API_KEY" \
  -d "action=add" \
  -d "service=1" \
  -d "link=https://youtube.com/..." \
  -d "quantity=1000"
Success Response 200 OK 
{
  "order": 12345
}
